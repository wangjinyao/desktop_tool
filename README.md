# 试用electron

## 安装开发环境

---

* 安装node：
* 检查node：`node -v`
* 安装npm：
* 检查npm：`npm -v`
* 安装electron：`npm i --save-dev electron`
* 检查electron：

```cmd
>npm ls electron
desktop_tool@1.0.0 D:\open_source_code\desktop_tool
`-- electron@11.2.2
```

注意：如果安装不了，尝试单个安装卡住的模块。如果有版本号就说明安装成功


## 运行

---

启动命令：`npm start`
